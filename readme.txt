# 🛍️ Shopping App

A simple and modern Shopping App built using Android (Java) and Firebase. This project allows users to browse products, view product details, and make purchases. Firebase is used for real-time database management and user authentication.

## 🚀 Features

- 👤 User login with Firebase OTP Authentication
- 🛒 Add to cart and checkout functionality
- 📦 View list of products with real-time Firebase integration
- 🔍 Search and filter products
- 📱 Clean and responsive user interface
- 🌙 Dark/Light theme support (optional)
- 🔐 Secure authentication
